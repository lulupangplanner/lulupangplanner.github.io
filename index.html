Housing Price Prediction

Completion Date: December 2020
Research Method: Machine Learning (Python)
Dataset:  79 attributes with a target variable sales price. 

I first conducted data cleaning and then applied four models - NumPy linear regression, support vector machine, random forest, and decision tree - to predict housing prices. The results of these four algorithms performed differently, among which NumPy linear regression had the highest R^2 and the lowest RMSE (Root-Mean-Square Error).
